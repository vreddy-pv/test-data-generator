# Test Data Generator

An independent agent that generates realistic test data based on an SQL schema file.

## Features

*   Parses SQL schema files ( statements).
*   Generates realistic data for various SQL data types using the Faker library.
*   Outputs the generated data as both SQL  statements and an Excel file.
